3. API数据流量的计算方法
数据流量的计算方法是滚动计算，从当前时间前推7*24小时，这段时间内的用量超过了限制，则需要过一段时间再用，等到按前面所说计算方法计算出来的用量不再超过限制的流量。 具体每个函数的流量限制，请联系客户经理咨询。

4. 辅助工具
如何查找程序中提示错误ErrorCode的的含义，及如何处理？
在使用API时，一定要注意ErrorCode的处理。API的所有函数都会返回一个ErrorCode，所以，在调用函数后，首先需要检查返回的ErrorCode是否为0，如果为0，表明函数正常运行，如果不为0，说明函数运行有异常错误，此时需要重新运行函数或者人工干预，以便解决遇到的问题。可以使用 代码生成器->工具->错误查询 查找错误码信息。
